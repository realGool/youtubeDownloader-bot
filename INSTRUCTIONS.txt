========================================
ИНСТРУКЦИЯ ПО УСТАНОВКЕ И ЗАПУСКУ БОТА
========================================

1. УСТАНОВКА ЗАВИСИМОСТЕЙ
-----------------------
Выполните в терминале:
pip install -r requirements.txt

Это установит:
- aiogram (библиотека для Telegram ботов)
- yt-dlp (библиотека для скачивания видео с YouTube)
- python-dotenv (для загрузки переменных окружения)
- aiofiles (асинхронная работа с файлами)


2. УСТАНОВКА FFMPEG
-----------------
FFmpeg необходим для конвертации видео в MP3.

Для macOS:
brew install ffmpeg

Для Linux (Ubuntu/Debian):
sudo apt update
sudo apt install ffmpeg

Для Windows:
Скачайте с https://ffmpeg.org/download.html
и добавьте в PATH


3. ПОЛУЧЕНИЕ ТОКЕНА БОТА
-----------------------
1. Откройте Telegram
2. Найдите @BotFather
3. Отправьте команду /newbot
4. Следуйте инструкциям
5. Скопируйте полученный токен


4. НАСТРОЙКА БОТА
---------------
Создайте файл .env в корне проекта:
BOT_TOKEN=ваш_токен_здесь

Например:
BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz


5. ЗАПУСК БОТА
------------
Выполните в терминале:
python bot.py

Если бот запустился успешно, вы увидите в консоли сообщения без ошибок.


6. ИСПОЛЬЗОВАНИЕ
---------------
1. Найдите вашего бота в Telegram
2. Отправьте /start
3. Выберите язык (Русский или Тоҷикӣ)
4. Выберите формат (MP4 видео или MP3 аудио)
5. Отправьте ссылку на YouTube видео
6. Для MP4 выберите качество (1080p/720p, 480p, или 360p)
7. Подождите пока файл скачается


7. РАЗРЕШЕНИЕ ПРОБЛЕМ
--------------------
Если бот не отвечает:
- Проверьте правильность токена в .env
- Убедитесь что установлен FFmpeg
- Проверьте подключение к интернету
- Убедитесь что установлены все зависимости

Если возникают ошибки при скачивании:
- Проверьте правильность ссылки на YouTube
- Убедитесь что видео доступно
- Проверьте что установлен FFmpeg


8. ОСТАНОВКА БОТА
----------------
Нажмите Ctrl+C в терминале
